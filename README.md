## About

Application check for given sequence if there's an element whose sum of elements on its left is equal to the sum on its right side.

Input:

```
2
3
123
4
1233
```

Output:

```
No
Yes
```

## Application usage

Compile project:

```
cmake .
make
./runchecksidessum
```

### Prerequisites

* CMAKE version >= 3.7.2
